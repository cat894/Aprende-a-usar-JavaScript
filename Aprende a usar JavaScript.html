Aprende a usar JavaScript V2 BN

Versión “BN”: más Bien explicada y con Nuevos ejemplos. Verás variables modernas, funciones, DOM, eventos, arrays/objetos, bucles, y mini demos interactivas.

1) Variables: let, const

const crea referencias que no cambian; let permite reasignar. Evita var en código moderno.

const PI = 3.1416;
let contador = 0;
contador = contador + 1; // OK

2) Funciones y flechas

function sumar(a, b) { return a + b; }
const restar = (a, b) => a - b;

3) DOM + Eventos (demo en vivo)

Aquí verás el saludo…
const input = document.getElementById('nombre');
const salida = document.getElementById('salida');
document.getElementById('btnSaludar').addEventListener('click', () => {
  salida.textContent = `¡Hola, ${input.value || 'mundo'}!`;
});

4) Arrays y objetos

const frutas = ['manzana', 'pera', 'banana'];
const persona = { nombre: 'Ana', edad: 20 };

frutas.push('uva');             // ['manzana','pera','banana','uva']
const mayus = frutas.map(f => f.toUpperCase());
const mayores = [{ edad: 18 },{ edad: 25 }].filter(p => p.edad >= 21);

5) Bucles: for...of / forEach

for (const f of frutas) { console.log(f); }
frutas.forEach((f, i) => console.log(i, f));

6) Mini-proyecto: contador + lista rápida

0
let n = 0;
const num = document.getElementById('numero');
document.getElementById('mas').onclick = () => { n++; num.textContent = n; };
document.getElementById('menos').onclick = () => { n--; num.textContent = n; };

const lista = document.getElementById('lista');
document.getElementById('agregar').onclick = () => {
  const t = document.getElementById('tarea').value.trim();
  if (!t) return;
  const item = document.createElement('div');
  item.className = 'card';
  item.textContent = `• ${t}`;
  lista.appendChild(item);
  document.getElementById('tarea').value = '';
};

7) Fetch (concepto básico)

fetch sirve para pedir datos a una API. Ejemplo de forma general (no llama a ningún servidor real aquí):

async function cargarDatos() {
  const res = await fetch('/api/datos'); // tu endpoint
  const json = await res.json();
  console.log(json);
}
